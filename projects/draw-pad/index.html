Draw Pad

Draw Pad is a simple drawing app coded in Python and Pygame. The app was designed to demonstrate the use of the Pyjsdl-ts library package. Using Transcrypt and Pyjsdl, the code can be converted to JavaScript for online deployment. The Usage of Pyjsdl with Transcrypt post describes the procedure. Draw Pad is released under the GPL v3 license.

Draw Pad can be downloaded to use or examine the Python code. Draw Pad app on the page was compiled to JavaScript with Transcrypt and Pyjsdl.
